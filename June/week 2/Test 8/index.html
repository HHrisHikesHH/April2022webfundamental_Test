Q. In how many ways can we loop in an array describe each with an example ?

Answer:
const array = [1,2,3,4,5];

1st->while loop:

This is the simple loop which works on the condition given after while keyword into () this loop need to initiate iteration variable and also incremented.


const i=0;
while(i<=array.length){
console.log(array[i]);
i++;
}

2nd->for loop:

This is the most common loop which is simpleto declare as we can initiate increment and add conditioninto same for declaration body.


for(const i=0;i<=array.length;i++){
console.log(array[i]);
}

3rd->forEach loop:

This is the most common loop which is simpleto declare as we can initiate increment and add conditioninto same for declaration body.


array.forEach(element => {
console.log(element);
});

4th->for-in loop:

This loop is used when we need to iterate the array using the key.


for (const key in array) {
const element = array[key];
console.log(element);
}

5th->for-of loop:

This loop is used when we need to iterate the array using the index which is similar to simple for loop.


for (const i of array) {
console.log(array[i]);
}
